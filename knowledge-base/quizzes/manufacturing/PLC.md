# Question 1

### What does PLC stand for?

- A. Process Logic Computer
- B. Programmable Logic Controller
- C. Plant Logic Controller
- D. Process Level Computer

**Correct Answer:** B

**Explanation:**

PLC stands for Programmable Logic Controller, an industrial computer used to automate manufacturing processes.

---

# Question 2

### What is the primary function of a PLC?

- A. Monitor employee attendance
- B. Control industrial machines and processes
- C. Create invoices
- D. Store finished products

**Correct Answer:** B

**Explanation:**

PLCs continuously monitor inputs, execute logic, and control outputs to automate industrial operations.

---

# Question 3

### Which component is considered the "brain" of a PLC?

- A. Input Module
- B. Output Module
- C. CPU
- D. Power Supply

**Correct Answer:** C

**Explanation:**

The CPU executes the control program, processes input data, and sends commands to output devices.

---

# Question 4

### Which device sends information to a PLC?

- A. Actuator
- B. Sensor
- C. Motor
- D. Pump

**Correct Answer:** B

**Explanation:**

Sensors detect process conditions and send signals to the PLC.

---

# Question 5

### What is the correct PLC Scan Cycle sequence?

- A. Update Outputs → Execute Program → Read Inputs
- B. Read Inputs → Execute Program → Update Outputs
- C. Execute Program → Read Inputs → Update Outputs
- D. Read Outputs → Execute Inputs → Store Data

**Correct Answer:** B

**Explanation:**

A PLC continuously reads inputs, executes the program, updates outputs, and repeats the cycle.

---

# Question 6

### Which PLC programming language is most commonly used in industries?

- A. Python
- B. Java
- C. Ladder Logic
- D. HTML

**Correct Answer:** C

**Explanation:**

Ladder Logic is widely used because it resembles traditional electrical relay circuits.

---

# Question 7

### Which of the following is an example of a digital input?

- A. Temperature Sensor
- B. Pressure Sensor
- C. Push Button
- D. Flow Meter

**Correct Answer:** C

**Explanation:**

A push button has only two states (ON/OFF), making it a digital input device.

---

# Question 8

### Which of the following is an example of an analog input?

- A. Emergency Stop Button
- B. Limit Switch
- C. Temperature Sensor
- D. Selector Switch

**Correct Answer:** C

**Explanation:**

Temperature changes continuously, so it is measured as an analog signal.

---

# Question 9

### In a stainless steel manufacturing plant, PLCs are commonly used to control:

- A. Electric Arc Furnace
- B. Rolling Mills
- C. Conveyor Systems
- D. All of the Above

**Correct Answer:** D

**Explanation:**

PLCs control almost every automated process in a modern steel plant.

---

# Question 10

### A PLC suddenly stops receiving signals from a proximity sensor. What should be checked first?

- A. Paint color of the machine
- B. Sensor wiring and power supply
- C. Employee attendance
- D. Warehouse inventory

**Correct Answer:** B

**Explanation:**

The first troubleshooting step is to verify sensor power, wiring, and input status before investigating the PLC program.
